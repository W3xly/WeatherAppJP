# WeatherAppJP

Aplikace zjišťuje aktuální počasí po zadání lokalizace. </br>
Případně zjišťuje počasí podle aktuální lokalizace. </br>

Při tvorbě aplikace jsem využil Swift 5 (UIKit) a Alamofire.

Je potřeba nainstalovat Pods.
